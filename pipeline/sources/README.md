# PRISMA Source Lists

Each file contains TMDB IDs, one per line. Lines starting with `#` are comments.

## Files
- `sight_and_sound_2022.txt` — BFI Sight & Sound 2022 Top 250 (generated by sight_and_sound_2022.py)
- `palme_dor_historical.txt` — Palme d'Or winners 1959–2023
- `oscar_international.txt`  — Oscar Best International Film winners 1993–2023
- `to_ingest.txt`            — Manual additions (ad hoc)

## Adding to queue
```bash
# Add a full source list
python pipeline/queue_manager.py --add-list pipeline/sources/palme_dor_historical.txt --source palme_dor
python pipeline/queue_manager.py --add-list pipeline/sources/oscar_international.txt --source oscar_intl
python pipeline/queue_manager.py --add-list pipeline/sources/sight_and_sound_2022.txt --source sight_and_sound

# Add a single film by title
python pipeline/queue_manager.py --add-title "Portrait of a Lady on Fire" --year 2019 --source letterboxd

# Check queue status
python pipeline/queue_manager.py --status

# Remove films already in Supabase from the pending queue
python pipeline/queue_manager.py --deduplicate
```

## Then run the agent
```bash
# Dry run first (no writes)
python pipeline/ingest_agent.py --batch 10

# Execute for real
python pipeline/ingest_agent.py --batch 10 --execute
```
